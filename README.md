¿Qué valor agregado tiene el uso de WebComponents?

Permite crear componentes reutilizables y organizados, facilitando la modularidad del código y su mantenimiento

¿De qué forma manipularon los datos sin recargar la página?

Se utilizó el método preventDefault() en el evento submit para evitar la recarga y se modificó el DOM dinámicamente con JavaScript

¿De qué forma validaron las entradas?

Se verificó que los campos no estuvieran vacíos y que los valores fueran numéricos antes de realizar el cálculo.

¿Cómo manejarían la escalabilidad futura?

Separando componentes, utilizando más WebComponents y aplicando una arquitectura modular que permita agregar nuevas funcionalidades sin afectar el sistema principal.
